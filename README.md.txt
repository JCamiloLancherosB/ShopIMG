Proyecto diseñado como tienda online, incialmente se empezaron a trabajar las páginas para vender ropa, luego se amplió
la visión para vender muchos productos tecnologicos que al paso del tiempo se irán agregando a la página, estos productos
vendrán con imagenes como los que ya están, pero algunos adicionalmente traeran herramientas de personalización incluídas
en el paquete del producto, esto lo podemos ver en la carpeta llamada "USB", donde encontraremos diferentes funcionalidades
que nos ayudarán a trabajar con la página web y el programa, con las funciones de dicha carpeta podremos mover, copiar, 
buscar, enlistar y todo para ofrecer un contenido personalizado al cliente.

En proceso de incluir React y nextjs, luego manejamos los datos con mongoDB e ire agregando nuevas funcionalidades 
con el tiempo.
